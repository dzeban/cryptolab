# Задание для Дениса и Михаила #

Итак, на этот раз задание сразу на двоих.
У нас есть gmp и криптографические объекты.


Так вот, необходимо реализовать набор классов на языке С++ для работы с криптографическими объектами, такими как: поля, кольца, группы, полиномы. В каждом из этих объектов, если необходимо, возможно выделение подобъектов таких как: полугруппы, перестановки и т.д.

Таким образом реализованный класс объекта должен позволять:
  1. создать объект с определенными параметрами
  1. выполнять над объектом определенные для него операции
  1. уничтожить объект

Операции должны быть реализованы в виде методов соответствующих классов. Класс должен соответствовать какому либо объекту. Для реализации арифметических операций внутри классов необходимо использовать gmp.
